Raneto

Markdown powered Knowledgebase for Node.js

Raneto is an open source Knowledgebase platform that uses static Markdown files to power your Knowledgebase.

Simple

Create and manage an entire knowledgebase using your favorite text editor.

Flat

Raneto is a "flat file" CMS, meaning no database problems, no MySQL queries, nothing.

Fast

Node is fast + Raneto is seriously lightweight + Raneto has no database = super fast.

Other Features

  • Mustache for easy templating
  • Responsive default template (powered by Bootstrap)
  • Highlight.js code syntax highlighting
  • Full-text search powered by Lunr
  • GitHub Flavored Markdown
